These template files contained in the \Load\AspenPlus directory will help users prepare Aspen Plus simulations for import into ICARUS Process Evaluator.
AspenIPE template IP.apt is for Inch-Pound unit projects.
AspenIPE template MET.apt is for Metric unit projects.

-------------------------------------------------------------------------

Procedure

1. Open your existing simulation in Aspen.
	File>>Import -> AspenIPE template MET.apt  OR   AspenIPE template IP.apt

2. Run simulation
	File>>export -> ReportName.rep

3. Import the new ".rep" file into IPE.

-------------------------------------------------------------------------

Note:

This template file was generated by creating a blank simuation in
Aspen Plus, following the steps in The Aspen ICARUS Process Evaluator 
User's Guide for Preparing Simulation Reports (Aspen Plus Versions 10.0
and greater, as listed on pg 2-13), and saving as an Aspen Plus
Template file (*.apt).
